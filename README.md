# Sangam Portfolio — Local Setup (npm)

TanStack Start + Vite + Tailwind v4.

## Run locally
```bash
npm install
npm run dev
```
Open the printed local URL (usually http://localhost:5173).

## Build
```bash
npm run build
npm run start
```

Note: `bun.lock` / `bunfig.toml` are leftover from the Lovable workspace and can be deleted — npm will create its own `package-lock.json`.
